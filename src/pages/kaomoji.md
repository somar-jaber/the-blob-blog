**Kaomoji** (顔文字) is a Japanese term for **emoticons** or **"face characters"** created using text characters, punctuation marks, and symbols, which are used to express emotions, actions, and concepts in digital communication.

The word itself is a combination of two Japanese words:
* ***Kao*** (顔) meaning "face"
* ***Moji*** (文字) meaning "character"

### Common Examples

| Kaomoji | Meaning |
| :--- | :--- |
| **(\^\_\^)** | Happy, smiling |
| **(T\_T)** | Crying, sadness |
| **(o\_o)** | Surprised, shocked |
| **(\*´ω｀\*)** | Blushing, cute embarrassment |
| **¯\\\_(ツ)\_/¯** | Shrugging (I don't know, whatever) |
| **(╯°□°）╯︵ ┻━┻** | Flipping a table (Anger, frustration) |
