## **Kosmik** (paid)
A Browser acts like a whiteboard imbedded with AI. It is Amazing for Research either for science or for arts.
IT is paid but has a free 1-week free trailer.

## **Arc**
It is an elegant free browser with a new design concept. It is my default browser, it built on Chromium. 

## **Comet** (paid)
A browser from Perplexity. it allows the AI to take over the browser and make actions. It is paid now but used to be free. 

## **GPTAtlas** (paid)
Like Comet but this is the original author for the idea. and it is a lot more capable than Comet but also a lot 
more expensive.

## **Obsidian**
The powerful offline free replacement for Notion. 
It has a mobile edition. but the Sync future is paid(this is the only paid thing in the app).

## **Anytype**
My default knowledge base. It still in development but it is safe enough now to start using it.
It is a local-first encrypted knowledge base.
I prefer it more that Obsidian for now and it has a mobile edition too.

## **SiYuan**
A free offline Chinese replacement for Notion.
It has a Mobile edition too.

## **DeepL**
AI translation app. very useful for scientific use & context. It has a PC version which adds 
a very useful futures like direct translating for text using short cuts. It also checks and improves texts
but this is yet not supported for all languages.

## **PlayNet**
For Games Lovers, this software is a video game library manager, it is very useful cool.
It has themes which some of them gives the feel of PS themes.
My best theme is PlayniteModernUI.

## **Net Limiter**  
Very powerfule and user friendly app to strict and control network access for all apps and services.

## **Keyboard Lock**
If you study on your laptop and don't have a much space in your table so absolutly
you want to lock you keyboard so you can put your notebook on it to write without
worrying from doing something wrong with your PC.


## **Wispr Flow**
Voice-to-text AI agent, you can talk and it will handle the input in any app or input field 
Very useful for journalists and for daily tasks. It has a ver generous free plane and very fast performance 
and accuracy even for the hard languages like Arabic and Russian. It works online but it feels like it is working offline 
because of its speed and low bandwidth.
